## 🎯 Project Purpose

The primary objective of this project is to analyze historical data from the global 
video game industry to identify key patterns that drive commercial success. By 
evaluating platform lifecycles, regional sales trends, and the influence of critic 
and user reviews, this analysis provides data-driven insights for the online store **ICE**. 

The ultimate goal is to forecast top-performing products and optimize marketing 
strategies for the upcoming year. Through rigorous data cleaning and statistical 
hypothesis testing, this project reveals which variables—such as genre, platform 
dominance, or ESRB ratings—truly impact profitability across North America, 
Europe, and Japan.
